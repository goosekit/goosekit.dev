Good production fit

  • Internal tools, scripts, agents, or product flows that need JSON, hash, UUID, or password utility endpoints.
  • You already tested the free endpoint and now need higher limits, a production-friendly setup, or a clearer support path.
  • You can share estimated monthly volume and whether the use case is commercial or client-facing.

Probably not a fit

  • You only need a few manual calls while debugging locally.
  • You need a formal SLA, procurement process, or enterprise contract immediately.
  • You want a broad API marketplace rather than a small utility endpoint pilot.

What to include in a production request

A useful request is short but specific. The fastest path is to say which endpoint you need, what will call it, and what volume would make the free testing tier too small.

  • Endpoint: JSON formatter, hash generator, UUID generator, password generator, or the adjacent endpoint you need next.
  • Monthly volume: expected requests per month and whether traffic is steady or bursty.
  • Where it runs: internal tool, script, agent workflow, backend job, browser app, or customer-facing product.
  • Commercial context: personal, internal business, client-facing, or paid product integration.
  • Timing: when you need production access and what breaks if the endpoint is unavailable.

🔑 Authentication

None required for the testing tier. Just call the endpoints directly from your app. Rate limited to 100 requests/day per IP address. CORS is enabled, so you can call these endpoints straight from a browser. For production workloads, request access before depending on the free tier.

curl -X POST https://goosekit-json.goosekit.workers.dev/format \
  -H "Content-Type: application/json" \
  -d '{"json":"{\"hello\":\"world\"}"}'

Endpoints

Use POST for JSON formatting and hashing. Use GET for UUID and password generation.

POST JSON Formatter

Format, minify, and validate JSON data.

POST https://goosekit-json.goosekit.workers.dev/format
Content-Type: application/json

{
  "json": "{\"name\":\"goosekit\",\"version\":1}",
  "indent": 2
}

POST Hash Generator

Generate MD5, SHA-1, SHA-256, SHA-512 hashes.

POST https://goosekit-hash.goosekit.workers.dev/hash
Content-Type: application/json

{"text": "hello", "algorithm": "SHA-256"}

GET UUID Generator

Generate v4 UUIDs. Supports batch generation.

GET https://goosekit-uuid.goosekit.workers.dev/generate?count=5

GET Password Generator

Generate cryptographically secure random passwords.

GET https://goosekit-password.goosekit.workers.dev/generate?length=32&symbols=true
