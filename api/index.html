🔑 Authentication

None required. Just call the endpoints directly from your app. Rate limited to 100 requests/day per IP address. CORS is enabled, so you can call these endpoints straight from a browser.

curl -X POST https://goosekit-json.goosekit.workers.dev/format \
  -H "Content-Type: application/json" \
  -d '{"json":"{\"hello\":\"world\"}"}'

Endpoints

Use POST for JSON formatting and hashing. Use GET for UUID, password, and lorem generation.

POST JSON Formatter

Format, minify, and validate JSON data.

POST https://goosekit-json.goosekit.workers.dev/format
Content-Type: application/json

{
  "json": "{\"name\":\"goosekit\",\"version\":1}",
  "indent": 2
}

POST Hash Generator

Generate MD5, SHA-1, SHA-256, SHA-512 hashes.

POST https://goosekit-hash.goosekit.workers.dev/hash
Content-Type: application/json

{"text": "hello", "algorithm": "SHA-256"}

GET UUID Generator

Generate v4 UUIDs. Supports batch generation.

GET https://goosekit-uuid.goosekit.workers.dev/generate?count=5

GET Password Generator

Generate cryptographically secure random passwords.

GET https://goosekit-password.goosekit.workers.dev/generate?length=32&symbols=true

📊 Rate Limits

Free tier: 100 requests/day per IP address across all endpoints. Rate limit resets daily at midnight UTC.

When you hit the limit, you'll receive a 429 Too Many Requests response with a Retry-After header.

For production use, higher-volume internal tools, or partner-style access, request production access and include your expected volume plus endpoint needs.
