Developer tools

Goosekit is strongest when a developer needs a quick utility page during a real workflow, like formatting JSON, decoding a JWT, testing a regex, or converting cURL into code.

Core developer workflow

  • JSON FormatterFormat, validate, minify, and inspect JSON.
  • Regex TesterTest patterns with live match feedback.
  • JWT DecoderInspect JSON Web Tokens instantly in the browser.
  • Base64 ToolEncode and decode text or images.
  • Hash GeneratorGenerate MD5, SHA-256, and SHA-512 hashes.
  • SQL FormatterBeautify SQL queries for readability.

Code and text helpers

  • cURL ConverterTurn cURL commands into production code snippets.
  • Markdown PreviewerWrite and preview Markdown live.
  • Diff CheckerCompare text side by side.
  • JSON to TypeScriptGenerate interfaces from JSON samples.
  • JSON to YAMLConvert between JSON and YAML.
  • URL Encoder / DecoderEncode and decode URL strings quickly.

Design and image tools

These are useful for quick visual asset work, UI polish, and browser-side image adjustments without heavy software.

Visual asset tools

  • Screenshot BeautifierAdd gradients, framing, spacing, and polish to screenshots.
  • Favicon GeneratorCreate favicons from text, emoji, or images.
  • OG PreviewPreview Open Graph cards before publishing.
  • Color PickerPick colors and convert between color formats.
  • Color Palette GeneratorGenerate palettes and harmonious color sets.
  • CSS Gradient GeneratorCreate gradients visually and copy the CSS.

Image workflow tools

  • Image CompressorReduce file size in the browser.
  • Image ResizerResize images quickly without opening heavier apps.
  • Image CropperCrop images with control over framing.
  • Background RemoverRemove image backgrounds for fast asset cleanup.
  • SVG to PNGExport SVG assets into PNG files.
  • Image to PDFCombine image files into a simple PDF.
  • PDF EditorReorder, rotate, and delete pages, then download the updated PDF.

Writing, utility, and productivity tools

Goosekit also covers small everyday workflows that usually do not justify a full app install.

Text and content utilities

  • Word CounterCount words, characters, and reading time.
  • Text Case ConverterSwitch between upper, lower, title, and sentence case.
  • Lorem Ipsum GeneratorGenerate placeholder copy fast.
  • HTML Entity Encoder / DecoderHandle special characters for HTML output.
  • Online NotepadUse a simple browser-based scratchpad.
  • Meta Tag GeneratorCreate meta tags for HTML pages.

General utility tools

  • UUID GeneratorGenerate UUIDs quickly in the browser.
  • Timestamp ConverterConvert Unix timestamps and human dates.
  • Cron Expression ParserTranslate cron expressions into plain English.
  • Percentage CalculatorRun everyday percentage calculations.
  • Unit ConverterConvert common units without leaving the tab.
  • QR Code GeneratorCreate QR codes for links and quick sharing.
