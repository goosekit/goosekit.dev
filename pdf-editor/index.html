Honest limitations

This is a practical browser editor, not a full desktop PDF suite. Here is what it does not do, so you know before you start.

  • Existing embedded text is not rewritten. Edits are added as overlays and flattened on export. To rewrite the original underlying text in a PDF, you need a desktop tool like Adobe Acrobat.
  • No OCR for scanned PDFs. If your file is a scan with no text layer, you can still annotate and sign it, but you cannot select or edit the scanned text.
  • No form-field editing. You can place text on top of form fields, but you cannot create new interactive AcroForm fields here.
  • Password-protected PDFs are not supported. Decrypt the file in your PDF reader first, then bring it here.
  • Very large PDFs may be slow. Everything runs in your browser, so a several-hundred-page file with images can take noticeable time to render and export.

Frequently asked questions

Quick answers about the editor, privacy, and what it can and cannot do.

Is this PDF editor really free?

Yes. The editor is free, has no signup, no watermark, and no per-file limit. It runs entirely in your browser.

Does my PDF get uploaded to a server?

No. Your PDF is opened locally in your browser using pdf.js and pdf-lib. The file never leaves your device. You can verify by opening DevTools → Network and watching no request go out when you load and edit a file.

Can I edit existing text inside a PDF, like in Adobe Acrobat?

No. This tool adds new text, drawings, highlights, images, and signatures as overlays on top of the page. The original embedded text and images are not rewritten. If you need to rewrite the underlying text in an existing PDF, a desktop tool like Acrobat is the right choice.

How do I sign a PDF online with this tool?

Open your PDF, click the Sign tool, and either draw your signature with the mouse or trackpad or upload a PNG/JPG of an existing signature. Click anywhere on the page to place it, then drag and resize. Click Download edited PDF to get a flattened, signed PDF.

How do I reorder, rotate, or delete pages?

Each page in the left strip has up, down, rotate (90°), and delete controls. The exported PDF reflects the new order, the rotation, and any pages you removed. The action is non-destructive until you download — clear the file to start over.

What format does it export?

A standard PDF file. Annotations are flattened into the page content so the output opens cleanly in any PDF reader, browser, or email attachment, with no missing fonts or extra layers.

Does it work on mobile?

Yes for light edits — annotations, signatures, and page-level reorder/rotate/delete work on phones and tablets. For heavy editing, a laptop with a real cursor is much more comfortable.

How is this different from PDF Merge and Image to PDF?

PDF Merge combines several PDF files into one. Image to PDF turns photos or screenshots into a PDF. This editor is for edits to a single existing PDF — annotations, signatures, and page rearrangement — and works best after you have already merged or generated the file.
