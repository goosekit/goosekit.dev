📖 Permission Reference

r (4)
Read — View file contents or list directory
w (2)
Write — Modify file or add/remove directory entries
x (1)
Execute — Run file or traverse directory
s (setuid)
Set User ID — Execute as file owner (on owner-x)
s (setgid)
Set Group ID — Execute as file group (on group-x)
t (sticky)
Sticky Bit — Only owner can delete in directory
