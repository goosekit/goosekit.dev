Open Graph meta tags checklist and preview

Generate the tags you need, then preview how the page appears in social feeds before you ship a landing page, article, product page, or launch post.

Generate meta tagsPreview social cards

Checklist

  • Write a title that fits the search result and social card.
  • Add a concise meta description with the page's real promise.
  • Set canonical URL, robots, Open Graph title, description, image, and URL.
  • Preview the page before sharing it on Discord, Slack, X, LinkedIn, or Product Hunt.

Common misses

  • Missing 1200x630 social image.
  • Different title promises between SEO and Open Graph.
  • Placeholder descriptions left in production.
  • Wrong canonical URL after moving a page.
