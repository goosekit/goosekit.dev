Ship It Kit is an independent starter with a different stack focus — Next.js + Supabase + Stripe specifically, with a smaller scope and lower price point. It is not a fork of ShipFast.
What exactly do I get with Ship It Kit?
A clean Next.js 15 starter repo with Supabase email auth, session-aware app gating, Stripe Checkout, customer portal, webhook handlers, billing settings page, password reset, and a typed database layer. The proof today is the app-level auth + billing flow, not a pretend middleware claim.
Can I use Ship It Kit with an AI coding tool?
Yes. The smaller scope is intentional — it keeps the repo easy for Cursor, Claude, and Copilot to navigate and edit without wading through a large framework.
What if it is the wrong fit after buying?
30-day money-back guarantee, no questions asked. Instant repo access after purchase so you can evaluate the starter immediately.
Does Ship It Kit work with any Stripe billing model?
It is set up for one-time and subscription billing via Stripe Checkout, with webhook handling for subscription lifecycle events and a customer portal for self-service management.
